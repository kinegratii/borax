# v4.1.0发布日志

> 发布时间：2024年1月31日



## 1 项目开发

从 4.1.0 开始，Borax 在项目开发构建上有重大变更，具体包括：

- **Borax不再支持 python3.7和python3.8，最低版本为3.9**
- 本地开发环境更新至3.11
- 使用 *pyproject.toml* 取代原有的 *setup.py* 和 *setup.cfg* 文件。
- 更新大量开发依赖库( *requirements_dev.txt* )的版本。



## 2 日历应用

Borax 提供了一个基于 tkinter 的日历应用程序，该日历应用包含了一些常见的功能：

- 万年历显示
- 日期计算相关工具
- 查看节日、节气、干支信息
- 创建和导出节日源
